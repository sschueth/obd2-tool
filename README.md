## obd2-tool
This project is to create a user interface with the ELM327 OBD-II connector.

Features:
* On-board diagnostics
  * Read up to 1000 fault codes
  * Clear fault codes
* Performance Validation
  * 1/4 mile test
  * 0 to 60 test
* Condition Monitoring
  * Temperatures
  * Pressures
  * Oil levels
